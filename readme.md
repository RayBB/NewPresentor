# Ubiquitous Presenter 2.0
A program designed to bring UCSD's Ubiquitous Presenter into the modern age.


## Setup
```bash
git clone https://github.com/RayBB/NewPresentor.git
cd NewPresentor
npm install
nodemon server.js
```
